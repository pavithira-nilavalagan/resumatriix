# 📄 Resume Matrix - AI Resume Builder

A professional resume builder with AI-powered PDF parsing using Google Gemini.

## ✨ Features

- 🎨 Create professional resumes from scratch
- 📤 Upload and auto-parse existing PDF resumes with AI
- 🔒 100% client-side, no data storage, complete privacy
- 📱 Fully responsive design
- 🎯 ATS-optimized resume templates

## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, Tailwind CSS
- **Backend**: Node.js, Express.js
- **AI**: Google Gemini API
- **PDF**: pdf.js

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Google Gemini API Key
